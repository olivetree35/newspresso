# Base Skill (기본 스킬)

이 스킬은 사용자의 요청을 받아 터미널에서 가시적으로 명령을 실행하고 그 결과를 한국어로 보고합니다.

## 지시사항 (Instructions)

1. **사용자 의도 파악 (Understand Intent)**:
   - 사용자가 무엇을 실행하고 싶은지 명확히 파악하십시오.
   - 불명확한 경우 실행 전 확인 질문을 하십시오.

2. **터미널 실행 (Terminal Execution)**:
   - 반드시 ** 터미널 명령**을 생성하여 ai 작업현황을 볼수 있게 실행하십시오.
   - **Windows PowerShell** 환경임을 고려하여 호환되는 명령어를 사용하십시오.
   - **한글 출력 깨짐 방지**:
     - Python 스크립트 실행 시 명령어 앞에 `$env:PYTHONIOENCODING='utf-8';`를 붙이거나, 코드 내에서 `sys.stdout.reconfigure(encoding='utf-8')`을 사용하도록 하십시오.
   - 긴 작업의 경우 `tqdm`이나 주기적인 `print` 문을 활용하여 진행 상황을 알리도록 코드에 포함하십시오.

3. **결과 보고 및 분석 (Korean Response)**:
   - 실행 결과(Output)와 분석 내용(Analysis)은 반드시 **한국어**로 작성하십시오.
   - 코드 내 주석(Comments)도 가능한 **한국어**로 작성하여 사용자가 이해하기 쉽게 하십시오.
   - 성공 여부와 관계없이 실행된 명령의 의도와 결과를 명확히 설명하십시오.

## 제약조건 (Constraints)

- **출력 제한**: 실행 결과가 50줄을 넘어가면 전체를 출력하지 말고 핵심 내용 위주로 요약해서 보여주십시오.
- **에러 핸들링**: 에러 발생 시 단순 번역이 아닌, 로그를 분석하여 원인을 파악하고 **한국어로 구체적인 해결책**을 제안하십시오.
- **안전성**: 시스템 설정 변경, 파일 삭제, 외부 네트워크 전송 등 위험한 작업은 반드시 `SafeToAutoRun`을 `false`로 설정하고 사용자의 승인을 받으십시오.

## 예시 (Examples)

### 예시 1: 파일 목록 확인
**사용자**: "현재 폴더 파일 리스트 보여줘"
**에이전트 행동**:
1. 터미널 명령 생성: `Get-ChildItem -Force` (또는 `ls -Force`)
2. 실행: `run_command`
3. 결과 해석: "현재 디렉토리(`D:\...`)에는 다음과 같은 파일들이 있습니다..." (한국어 설명)

### 예시 2: 파이썬 스크립트 실행
**사용자**: "데이터 수집 스크립트 실행해줘"
**에이전트 행동**:
1. 인코딩 설정 제안: "한글 출력을 위해 utf-8 인코딩을 설정하여 실행합니다."
2. 명령 실행: `$env:PYTHONIOENCODING='utf-8'; python collector.py`
3. 결과 보고: 스크립트의 출력 로그를 분석하여 성공/실패 여부를 한국어로 요약 보고.
